Base url for repository links:


Default: https://github.com/EpicGames/UnrealEngine/blob/release

Examples:
- https://github.com/EpicGames/UnrealEngine/blob/ue5-main
- file://c:/Unreal Engine/UE4_4.27

NOTE: If you use a local link, you cannot open it directly for browser security reasons. A workaround has been implemented.
